# GEMINI.md

## Project
Premium bilingual personal website for Tomefy.

## Brand
Tomefy = AI Automation Engineer + AI Agent Engineer + Entrepreneur.

## Goals
- Build a premium, editorial, bilingual portfolio
- Support French and English
- Make the site feel high-end, modern, and distinctive
- Avoid generic developer portfolio aesthetics

## Language rules
- Every visible text must exist in FR and EN
- Use locale-based routes
- Keep translation keys consistent
- Do not hardcode copy inside components if it varies by language

## Visual direction
- Dark premium background
- Non-classic accent color
- Minimal but expressive motion
- Elegant typography
- Strong whitespace
- Editorial layout

## Pages
- Home
- Work
- Lab
- Entrepreneurship
- Notes
- About
- Contact

## Content strategy
- FR and EN versions for all core pages
- Notes can be MDX
- Case studies should be concise and credible
- Avoid clutter

## Technical rules
- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- Clean reusable components
- Accessible semantic HTML
- SEO-ready metadata

## Quality bar
- Premium feel
- Fast loading
- Mobile-first
- Consistent spacing
- Great typography
- No unnecessary complexity
